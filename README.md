# Namaste-app
A video conference application built using Python Flask, NodeJS, and PeerJS. It is basically a Google Meet clone.

**Build Using:**
* Python-Flask
* SocketIO
* NodeJS
* PeerJS
* HTML
* CSS
* JavaScript
* Jquery
